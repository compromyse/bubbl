# Bubbl

An Operating System for The x86 Architecture.

## Building

### Dependencies:

> nix with flakes enabled and (optionally) direnv

After entering the devshell,

```sh
source env.sh
b
cm ..
m
```

## Running

```sh
m run # or run-headless or run-gdb
```
