# Bubbl

An Operating System for The x86 Architecture.

## Building

### Dependencies:

```sh
sudo apt install cmake ninja-build aria2 xorriso grub-common qemu-system-x86
./scripts/fetch-toolchain.sh

source env.sh
b
cm ..
m
```

## Running

### Dependencies:

- QEMU (qemu-system-i386)

```sh
mr
```
