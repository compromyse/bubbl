# Bubbl

An Operating System for The x86 Architecture.

## Building

### Dependencies:

- CMake
- Ninja
- Aria2 (for fetch-toolchain.sh)
- GRUB (grub-mkrescue)
- xorriso

```sh
./scripts/fetch-toolchain.sh

source env.sh
b
cm ..
m
```

## Running

### Dependencies:

- QEMU (qemu-system-i386)

```sh
mr
```
